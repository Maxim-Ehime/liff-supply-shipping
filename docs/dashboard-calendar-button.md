# ダッシュボード日付選択ボタンの設定手順

ダッシュボードの `A3` は、カスタムカレンダーを開くためのボタン置き場です。
Apps Script から図形ボタンの作成とスクリプト割り当てまでは自動化できないため、初回だけスプレッドシート上で以下を設定します。

## 前提

- `clasp push` 後、スプレッドシートを再読み込みします。
- メニューの `ダッシュボード > 初期作成` を実行します。
- `A3:B3` と `N3:N102` には警告付き保護が設定されます。編集自体は可能ですが、誤編集時に警告が出ます。

## 図形ボタンの作成

1. ダッシュボードシートを開きます。
2. メニューから `挿入 > 図形描画` を選択します。
3. 角丸四角形などを作成し、テキストを `日付を選択` にします。
4. 図形を保存して閉じます。
5. 追加された図形を `A3` の上に重ねます。
6. 図形右上のメニューから `スクリプトを割り当て` を選択します。
7. 関数名として `showDashboardCalendar` を入力します。

## 運用

- 図形ボタンをクリックすると、複数日付選択カレンダーが開きます。
- 選択結果の表示は `B3` に入ります。
- 実際の選択日付は非表示の `N3:N102` に保存されます。
- `L:M` は既存の「済」同期用メタ情報なので触らないでください。
- 図形がずれたり消えたりした場合は、メニューの `ダッシュボード > 日付を選択` からも同じカレンダーを開けます。
